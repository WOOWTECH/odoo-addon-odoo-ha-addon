# description

取回 entry 清單。

entry 是一個 integration 的初始化的實體。以 "domain": "glances" 來說，一個 glances 的 entry (id: 01KBA31BNXAWX73PQ3W00QHXZG)，它會初始化產生一個 device (id: 2ac79e2bf20b7e58eb2113d842ef6ef3)，這個 device 裡面包含多個 entities，device entities 可以從 config/entity_registry/list 或是 search/related 找到 entity id 清單。

# example

Send

```
{"type":"config_entries/get","id":48}
```
