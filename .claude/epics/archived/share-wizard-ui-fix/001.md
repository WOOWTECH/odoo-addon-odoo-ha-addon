# Task: 修改 XML 視圖定義，移除外層 group 標籤

## Description

修復 Entity Share Wizard 中「Already shared with」提示訊息區塊的 UI 跑版問題。移除外層的 `<group>` 標籤，將 `invisible` 屬性直接放在 `<div>` 上。

## Acceptance Criteria

- [ ] 移除 `views/ha_entity_share_wizard_views.xml` 第 20 行的 `<group>` 標籤
- [ ] 移除第 27 行的 `</group>` 結束標籤
- [ ] 將 `invisible="existing_share_count == 0"` 屬性加到 `<div>` 標籤上
- [ ] XML 語法正確，無錯誤

## Technical Details

**檔案**：`views/ha_entity_share_wizard_views.xml`
**行號**：20-27

**修改前：**
```xml
<group invisible="existing_share_count == 0">
    <div class="alert alert-info" role="status">
        <i class="fa fa-info-circle me-2" title="Info"/>
        <strong>Already shared with:</strong>
        <span><field name="existing_share_users" nolabel="1"/></span>
        (<field name="existing_share_count" nolabel="1"/> users)
    </div>
</group>
```

**修改後：**
```xml
<div class="alert alert-info" role="status" invisible="existing_share_count == 0">
    <i class="fa fa-info-circle me-2" title="Info"/>
    <strong>Already shared with:</strong>
    <span><field name="existing_share_users" nolabel="1"/></span>
    (<field name="existing_share_count" nolabel="1"/> users)
</div>
```
